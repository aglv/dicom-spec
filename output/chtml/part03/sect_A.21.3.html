A.21.3.1 Acquisition Context Module

The Defined TID for Acquisition Context Sequence (0040,0555) is TID 3470 “NM/PET Acquisition Context”, which includes description of the cardiovascular rest or stress state, and the blood glucose measurement.

The Acquisition Context information may be entered during acquisition, or obtained from the Modality Worklist using information supplied in the Protocol Context, using TID 15101 “NM/PET Protocol Context”.
