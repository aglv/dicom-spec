Note

  1. Within the RT Dose IOD, the RT Dose Module supports 2D and 3D dose grids, and the RT DVH Module supports dose-volume histogram data. They are not mutually exclusive: they may be included in a single Instance of the object. Product Conformance Statements should clearly state which of these mechanisms is supported and under what conditions.

  2. The RT Dose IOD has been defined as a composite IOD, separate from the RT Plan IOD. This has been done for the following reasons:

    • to allow for the multiplicity of possible dose calculations using beam models for the same basic plan,

    • to avoid undesirable transmission of large amounts of data with the treatment plan, and

    • to accommodate the fact that CT Simulation and other "beam geometry" generating devices that use the RT Plan IOD do not have or require access to this data, either for transmission or storage.

  3. The Audio Module (Retired) was previously included in this IOD but has been retired. See PS3.3-2004.

  4. The Overlay Plane Module, Multi-frame Overlay Module, Modality LUT Module, Structure Set Module, ROI Contour Module and RT Dose ROI Module (Retired) were previously included in this IOD. See PS3.3-2022d.
