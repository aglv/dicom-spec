C.8.34.2 Photoacoustic Acquisition Parameters Module

This section describes Photoacoustic Imaging parameters that are constant for the acquisition of all frames in this image.

Table C.8.34.2-1. Photoacoustic Acquisition Parameters Module Attributes

Attribute Name

Tag

Type

Attribute Description

Excitation Wavelength Sequence

(0018,9825)

1

Optical excitation wavelength(s) applied to the target. Acquired data from all input wavelengths in this sequence is processed to create the image. The processing may be described in Reconstruction Algorithm Sequence (0018,993D).

One or more Items shall be included in this Sequence.

>Excitation Wavelength

(0018,9826)

1

The wavelength in nm of the optical excitation pulse from the illuminator.

NO

(0022,0016)
