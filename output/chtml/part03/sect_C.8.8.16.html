C.8.8.16 Approval Module

Table C.8-52. Approval Module Attributes

Attribute Name

Tag

Type

Attribute Description

Approval Status

(300E,0002)

1

Approval status at the time the SOP Instance was created.

Enumerated Values:

APPROVED

Reviewer recorded that object met an implied criterion

UNAPPROVED

No review of object has been recorded

REJECTED

Reviewer recorded that object failed to meet an implied criterion

Review Date

(300E,0004)

2C

Date on which object was reviewed. Required if Approval Status (300E,0002) is APPROVED or REJECTED.

Review Time

(300E,0005)

2C

Time at which object was reviewed. Required if Approval Status (300E,0002) is APPROVED or REJECTED.

Reviewer Name

(300E,0008)

2C

Name of person who reviewed object. Required if Approval Status (300E,0002) is APPROVED or REJECTED.
