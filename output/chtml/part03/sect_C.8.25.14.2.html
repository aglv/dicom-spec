Attribute Name

Tag

Type

Attribute Description

Ophthalmic Axial Length

(0022,1019)

1

The axial length measurement, in mm.

See Section C.8.25.14.1.1.3 and Section C.8.25.14.1.1.4 for further explanation.

Ophthalmic Axial Length Measurement Modified

(0022,1140)

1

Whether or not the clinician intervened to modify the output of the device. For example by forcing it to select a different peak in the display.

Enumerated Values:

YES

NO
