A.80.2 Volume Rendering Volumetric Presentation State IOD

A.80.2.1 Volume Rendering Volumetric Presentation State IOD Description

The Volume Rendering Volumetric Presentation State IOD specifies information that defines a Volume Rendering presentation from volume data sets that are referenced from within the IOD.

It includes capabilities for specifying:

  1. spatial registration of the input data sets

  2. cropping of the volume data sets by a bounding box, oblique planes and segmentation objects

  3. the generation geometry of volume rendered reconstruction

  4. shading models

  5. scalar to RGB conversions

  6. compositing of multiple volume streams and one volume stream with segmentations

  7. clinical description of the specified view

  8. volume-relative and display-relative annotations, including graphics, text and overlays plus optional references to structured content providing clinical context to annotations.

  9. membership in a collection of related Volumetric Presentation States intended to be processed or displayed together

  10. the position within a set of sequentially related Volumetric Presentation States

  11. animation of the view

  12. reference to an image depicting the view described by the Volumetric Presentation State

The Volume Rendering Volumetric Presentation State IOD is used in three SOP Classes as defined in PS3.4:

  • one SOP Class allowing a single input volume and excluding segmentation cropping

  • one SOP Class allowing a single input volume with multiple input segmentations each with a distinct classification map

  • one SOP Class allowing multiple input volumes, segmentations, and classification maps
