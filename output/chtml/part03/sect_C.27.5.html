(0062,000F)

3

Sequence defining the specific property the surface point set represents.

Note

"Property" is used in the sense of meaning "what the surface represents", whether it be a physical or biological object, be real or conceptual, having spatial, temporal or functional extent or not. I.e., it is what the segment "is" (as opposed to some feature, attribute, quality, or characteristic of it, like color or shape or size).

Surface Point Presentation Value Data

(0080,0006)

3

Contains a vector of P-Values to assign a gray value to each point.

The number of values shall be equal to the value of Number Of Surface Points (0066,0015) in the Points Macro.

Each value shall correspond to the respective point in the Point Coordinates Data (0066,0016).

Surface Point Color CIELab Value Data

(0080,0007)

3

Contains a vector of color triplets to assign colors to each point.

The number of triplets shall be equal to the value of Number Of Surface Points (0066,0015) in the Points Macro.

The units are PCS-Values, and the value is encoded as CIELab. See Section C.10.7.1.1.

Each triplet shall correspond to the respective point in the Point Coordinates Data (0066,0016).
