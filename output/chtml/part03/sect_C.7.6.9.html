C.7.6.9 Frame Pointers Module

Table C.7-15 specifies the Attributes of the Frame Pointers Module.

Table C.7-15. Frame Pointers Module Attributes

Attribute Name

Tag

Type

Attribute Description

Representative Frame Number

(0028,6010)

3

The frame number selected for use as a pictorial representation (e.g., icon) of the Multi-frame Image.

Frame Numbers Of Interest (FOI)

(0028,6020)

3

Frame number(s) selected as frames of interest. A frame number may appear more than once.

Frame Of Interest Description

(0028,6022)

3

Description of each one of the Frame(s) of Interest selected in (0028,6020). If multiple Frames of Interest are selected and this Attribute is used, it shall contain the same number of values as are in Frame Numbers of Interest (0028,6020).

Frame of Interest Type

(0028,6023)

3

A Defined Term for each one of the Frame(s) of Interest (0028,6020) that identifies the significance of the frame. If multiple Frames of Interest are selected and this Attribute is used, it shall contain the same number of values as are in Frame Numbers of Interest (0028,6020).

Defined Terms:

HIGHMI

a frame acquired at the time of the high power pulse that destroys acoustic contrast

RWAVE

the frame closest to the R-Wave

TRIGGER

a trigger frame, for example a set delay from the R Wave

ENDSYSTOLE

the frame closest to end of systole, at the end of the T-wave

Note

  1. Frame numbers begin at 1.

  2. Frame of Interest Description is intended to indicate such frames as Systolic, Diastolic, Stenotic Artery, or trigger label.
