A.4 Structure of AIM Version 4 Instances

AIM instances are encoded in XML according to a schema generated from the AIM Model [AIM Model v4.2], which is defined in UML. Figure A.4-1 is a simplified view rather than the entire model, showing only those UML classes and attributes relevant to the transformations described in this Part.
