I.2 Store Instances Response Attribute Description

I.2.1 Warning Reason

Table I.2-1 defines the semantics for which the associated value shall be used for the Warning Reason (0008,1196):

Table I.2-1. Store Instances Response Warning Reason Values

Status Code (hexadecimal)

Status Code (decimal)

Meaning

Explanation

B000

45056

Coercion of Data Elements

The Studies Store Transaction (Section 10.5) modified one or more data elements during storage of the instance. See Section 10.5.3.

B006

45062

Elements Discarded

The Studies Store Transaction (Section 10.5) discarded some data elements during storage of the instance. See Section 10.5.3.

B007

45063

Data Set does not match SOP Class

The Studies Store Transaction (Section 10.5) observed that the Data Set did not match the constraints of the SOP Class during storage of the instance.

Additional codes may be used for the Warning Reason (0008,1196) to address the semantics of other issues.

In the event that multiple codes may apply, the single most appropriate code shall be used.
