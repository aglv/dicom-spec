10.4.3.3.6 Pixel Data Resource Payload

The payload for a Pixel Data Resource (see Section 10.4.1.1.6) shall contain all the Pixel Data of the resource. Pixel Data in a multipart response shall have a Content-Location header field that corresponds to the URI contained in the corresponding Element in the Metadata. The Pixel Data is the content of the Pixel Data (7FE0,0010), Float Pixel Data (7FE0,0008), or Double Float Pixel (7FE0,0009) Data Element in the top level Data Set, as defined in PS3.5, of the corresponding Instance(s).

Note

This does not include Pixel Data nested within an Icon Image Sequence or a private Data Element.
