HH.3 Defined Procedure Protocol Information Models

The Defined Procedure Protocol Information Models are based upon a one level entity:

  • Defined Procedure Protocol object instance.

The Defined Procedure Protocol object instance contains Attributes associated with the Procedure Protocol IE of the Composite IODs as defined in PS3.3 Information Object Definitions.
