JJ.4.2.2.3 C-GET SCP Conformance

An implementation that conforms to the Inventory GET SOP Class as an SCP shall support transfers against the Inventory Q/R Information Model using the C-GET SCP baseline behavior described for the Query/Retrieve Service Class (see Section C.4.3.3.1). The implementation shall support the Inventory Storage SOP Class (see Chapter GG) for the C-STORE sub-operations generated by the C-GET.
