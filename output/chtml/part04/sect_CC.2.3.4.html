CC.2.3.4 Status Codes

Table CC.2.3-3 defines the status code values that might be returned in a N-ACTION response. General status code values and fields related to status code values are defined for N-ACTION DIMSE Service in PS3.7.

Table CC.2.3-3. N-ACTION Response Status Values [for Subscribe/Unsubscribe to Receive UPS sEvent Reports]

Service Status

Further Meaning

Status Code

Success

The requested change of subscription state was performed

0000

Warning

Deletion Lock not granted.

B301

Failure

Failed: Specified SOP Instance UID does not exist or is not a UPS Instance managed by this SCP

C307

Failed: Receiving AE-TITLE is Unknown to this SCP

C308

Failed: Specified action not appropriate for specified instance

C314

Failed: SCP does not support Event Reports

C315
