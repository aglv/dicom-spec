CC.2.5.4 Status Codes

Table CC.2.5-4 defines the Status Code values that might be returned in a N-CREATE response. General Status Code values and fields related to Status Code values are defined for N-CREATE DIMSE Service in PS3.7.

Table CC.2.5-4. N-CREATE Response Status Values [for Create a Unified Procedure Step]

Service Status

Further Meaning

Status Code

Success

The UPS was created as requested

0000

Warning

The UPS was created with modifications

B300

Failure

Failed: The provided value of UPS State was not "SCHEDULED".

C309
