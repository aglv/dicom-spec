HH.4 DIMSE-C Service Groups

HH.4.1 C-FIND Operation

See the C-FIND Operation definition for the Section K.4.1.3.1 “"Worklist" Search Method”, and substitute "Defined Procedure Protocol" for "Worklist". The "Worklist" Search Method shall be used.

The SOP Class UID identifies the Defined Procedure Protocol Information Model against which the C-FIND is to be performed. The Key Attributes and values allowable for the query are defined in the SOP Class definitions for the Defined Procedure Protocol Information Model.

HH.4.1.1 Service Class User Behavior

No SOP Class specific SCU behavior is defined.

HH.4.1.2 Service Class Provider Behavior

No SOP Class specific SCP behavior is defined.
