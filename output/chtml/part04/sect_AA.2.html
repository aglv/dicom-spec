AA.2 Behavior of an SCP

For a device that is both an SCU and an SCP of the aforementioned Storage SOP Classes, in addition to the behavior for the Storage Service Class specified in Section B.2.2, the following additional requirements are specified for Ophthalmic Refractive Measurements SOP Classes:

  • An SCP of these SOP Classes shall support Storage Level 2 (Full) as defined in Section B.4.1.

Note

This requirement means that all Type 1, Type 2, and Type 3 Attributes defined in the Information Object Definition and Private Attributes associated with the SOP Class will be stored and may be accessed.
