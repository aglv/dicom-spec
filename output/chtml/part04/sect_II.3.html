II.3 Protocol Approval Information Models

The Protocol Approval Information Models are based upon a one level entity:

  • Protocol Approval object instance.

The Protocol Approval object instance contains Attributes associated with the Approval IE of the Composite IODs as defined in PS3.3 Information Object Definitions.
