JJ.4.2.2 SCP Conformance

JJ.4.2.2.1 C-FIND SCP Conformance

An implementation that conforms to the Inventory FIND SOP Class as an SCP shall support queries against the Inventory Q/R Information Model using the C-FIND SCP Behavior described for the Basic Worklist Management Service Class (see Section K.4.1.3 and Section JJ.3.1).

The implementation shall support the additional Attributes in the Response Identifier as specified in Table JJ.4-2 to support character set specification and access to Inventory SOP Instances.

Note

Compare Section C.4.1.1.3.2 for the similar specification of non-key Attributes that are to be returned in the Query/Retrieve C-FIND SOP Classes to support character set specification and access to the referenced SOP Instances.
