II.4 DIMSE-C Service Groups

II.4.1 C-FIND Operation

See the C-FIND Operation definition for the Basic Worklist Management Service Class (Section K.4.1), and substitute "Approval" for "Worklist". The "Worklist" Search Method shall be used.

The SOP Class UID identifies the Protocol Approval Information Model against which the C-FIND is to be performed. The Key Attributes and values allowable for the query are defined in the SOP Class definitions for the Protocol Approval Information Model.

II.4.1.1 Service Class User Behavior

No SOP Class specific SCU behavior is defined.

II.4.1.2 Service Class Provider Behavior

No SOP Class specific SCP behavior is defined.
