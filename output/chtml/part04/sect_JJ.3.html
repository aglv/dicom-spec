JJ.3 DIMSE-C Service Groups

JJ.3.1 C-FIND Operation

See the C-FIND Operation definition for the Basic Worklist Management Service Class (Section K.4.1), and substitute "Inventory" for "Worklist". The "Worklist" Search Method shall be used.

The C-FIND is to be performed against the Inventory Q/R Information Model (Section JJ.2). The Key Attributes and values allowable for the query are defined in the Inventory Q/R Information Model.
