CID 8131 Pathology Imaging Protocol

This Context Group is intended for use in the Scheduled Protocol Code Sequence (0040,0008) and the Performed Protocol Code Sequence (0040,0260).When so used, an Item with value (112703, DCM, "Whole Slide Imaging") may also include a Protocol Context Sequence (0040,0440) using TID 8010 “Slide Imaging Parameters”.
