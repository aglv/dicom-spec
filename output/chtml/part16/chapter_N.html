N Externally Defined Value Sets (Informative)

This annex identifies those Value Sets defined externally to the DICOM Standard that are referenced by the Standard. These value sets are reproduced here for reference only, and might not be the current version.

These value sets use codes from various coding schemes or code systems, as identified in Section 8.

N.1 HL7 Value Sets

HL7 Value Sets are reproduced with the permission of HL7 International. For the current version of HL7 Value Sets, see the HL7v3 Normative Edition (http://www.hl7.org/implement/standards/product_brief.cfm?product_id=186).
