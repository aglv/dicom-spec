CID 3227 QTc Measurement

This Context Group include both global and per lead corrected QT measurements specified in the ISO/IEEE 11073-10102 MDC nomenclature. Note that the MDC code for the per lead measurement is a base code for post-coordination with separately encoded lead identifiers. MDC also defines pre-coordinated codes that include both the measurement and the lead, which may be used in the same context as this Context Group; see the ISO/IEEE Standard.

While this Context Group includes distinct codes for the various QT correction algorithms, Templates using this Context Group may allow post-coordination using the QTc algorithm codes of CID 3678 “QT Correction Algorithm”.
