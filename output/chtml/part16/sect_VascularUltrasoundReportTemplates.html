Row 1

Previously, an Enumerated Value of (125100, DCM, "Vascular Ultrasound Procedure Report") was specified for the Concept Name. This code has been retired in favor of codes from external schemes, as well as permitting alternative codes to be used for more specific reports, e.g., for a Carotid Artery report. See PS3.16 2020a.
