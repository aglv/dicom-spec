Content Item Descriptions

Row 4

This row may be instantiated twice, once for resting state measurements, once for stress.

Row 15

The LVEF code specified in this row is defined in LOINC with method "imaging". LVEF measurement by ultrasound may also be encoded elsewhere in the Content Tree (e.g., in TID 3309 “Stress Echo Measurement Group”) with LOINC code 18043-0, which has method "ultrasound". It is recommended that such findings from the per-phase measurements be summarized here with the generic "LVEF by Imaging" concept code.
