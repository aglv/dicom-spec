TID 5016 LWH Volume Group

This Template is a container for a group of measurements that assess the size of an anatomical structure using a volume derived from perpendicular diameters.

Table TID 5016. Parameters

Parameter Name

Parameter Usage

$GroupName

The name of the volume group that is an anatomical structure

$Volume

Concept name of volume measurement

$Length

Concept name of length measurement

$Width

Concept name of width measurement

$Height

Concept name of height measurement

$DistanceUnits

Units of linear (height, weight, length) distance measurement

$VolumeUnits

Units of volume measurement

$FunctionalCondition

Functional condition present during measurement

$Method

Value for Measurement Method

$FindingType

Type of the finding
