Content Item Descriptions

Row 1

The maximum possible value of CAD Operating Point for this type of Detection Performed. No CAD Operating Point value recorded in the CAD Processing and Findings Summary sub-tree of the report for this type of Detection Performed shall exceed this value. The report may or may not contain Rendering Intent = "Presentation Optional" detections that are assigned the maximum value.

Row 2

A number indicating which of the CAD operating points is recommended by the creator of a CAD SR instance as the first operating point to be used when rendering the CAD SR instance contents. Subsequent changes to the displayed operating point are implementation dependent.
