9.3 MimeType

A data type whose values are Defined Terms that identify particular MIME content types. The syntax of the string defining a MIME content type is defined in [RFC2045]. Top level MIME content types are defined in [RFC2046]. MIME content types are drawn from the list managed by the Internet Assigned Numbers Authority (IANA) whose web site is at http://www.iana.org/assignments/media-types/, as described in [RFC2048].
