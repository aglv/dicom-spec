C.3.3.2 File Component IDs

Note

File Component IDs should be created using a random number filename to minimize File Component ID collisions as described in PS3.12. The FS-Updater should check the existence of a Component ID prior to creating that ID. Should an ID collision occur, the FS-Updater should try another ID.
